关于本项目

本系统是毕业设计项目，提出了基于分层最小能量点的音乐结构分析方法， 能够在不同粒度下自动识别乐谱中的分段结构（乐句、乐段等）。

创新点
  • 分层能量金字塔结构
  • 多粒度层级浏览器
  • 乐谱与曲线联动
技术栈
  • OSMD 乐谱渲染
  • Chart.js 数据可视化
  • Python 后端算法
